# 🎓 학습클리닉 통합관리 V9.6 (PWA)

충북학습종합클리닉 거점센터 업무지원을 위한 단일 HTML 기반 PWA입니다.

## 🚀 배포 (GitHub Pages)

```bash
git clone https://github.com/ekoredu-maker/jc-edu-clinic.git
cd jc-edu-clinic
# 본 산출물 4개 파일 복사 후
git add .
git commit -m "V9.6 PWA Release"
git push
```

GitHub 저장소 **Settings → Pages** 에서 `main` 브랜치 / `root` 를 선택하면 자동 배포됩니다.

배포 URL: `https://ekoredu-maker.github.io/jc-edu-clinic/`

## 📁 파일 구성

| 파일 | 용도 |
|---|---|
| `index.html` | 메인 앱 (단일 HTML, 모든 로직 포함) |
| `manifest.webmanifest` | PWA 매니페스트 |
| `sw.js` | Service Worker (오프라인 캐시) |
| `icons/icon-192.png` | PWA 아이콘 192px *(별도 준비 필요)* |
| `icons/icon-512.png` | PWA 아이콘 512px *(별도 준비 필요)* |

> 아이콘은 어떤 PNG여도 동작합니다. 없으면 로그만 찍히고 정상 작동합니다.

## 🔐 로그인

- 초기 비밀번호: `1111`
- SHA-256 해시로 저장

## ✨ 주요 기능 (V9.6)

### 🤖 매칭 엔진
- 시간 교집합 자동 계산 (분 단위, 최소 30분)
- 학생 1명 : 지원단 N명 복수매칭 지원
- 충돌(동일시간 중복신청) 시 수동조정 큐로 자동 이동
- 점수제: 교집합 슬롯 × 10 + 영역일치 × 5 + 우선순위 − 부담

### 📊 통계 5종 서식
분기별·월별 보고 서식을 버튼 한 번으로 엑셀 출력:
1. 지역거점별 지원 실적 (학교급별)
2. 지역별 지원현황
3. 방과후학습코칭 지원현황 (학년 × 9영역)
4. 치료기관연계 명단 + 집계
5. 난독증/경계선지능 교차피벗

### 📄 행정 서식
- 위촉장, 학습지도계획서, 방문일지, 지원단 명단 인쇄

### 🔒 개인정보 보호
- 이름 마스킹 4종: 표시 / OOO / 부분(홍OO) / 가명
