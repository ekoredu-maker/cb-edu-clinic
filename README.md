# 학습클리닉 통합관리 시스템 V10.3

제천거점 학습지원단 관리 업무를 오프라인 기반 모듈형 웹 애플리케이션으로 전환한 배포 패키지입니다.

## 핵심 변경

- V9.5 단일 HTML 구조를 `index.html / CSS / JS 모듈 / Service Worker` 구조로 분리
- IndexedDB 기반 저장소 사용
- 지원단 복수 가능시간, 학생 1·2희망 시간대, 매칭·활동기록·검증 흐름 분리
- 관리부, 방문일지, 진행계획서, 진행 확인서, 보고서 출력 템플릿 확장
- V10.3 통계 출력 강화
  - 월별 양식형 XLSX
  - 월별 검증형 XLSX
  - 분기별 양식형 XLSX
  - 분기별 검증형 XLSX
  - 표지, 양식형 실적, 지원단 지급산출, 세부활동내역, 자료점검 시트 포함
- 브라우저 오프라인 XLSX 내보내기 엔진에 기본 스타일, 병합셀, 열너비, 고정틀, 자동필터 지원 추가

## 실행 방법

1. GitHub 저장소에 이 폴더 전체를 업로드합니다.
2. Settings > Pages에서 `main / root`를 배포 소스로 지정합니다.
3. 발급된 GitHub Pages 주소로 접속합니다.
4. Chrome 또는 Edge 주소창의 앱 설치 버튼으로 PWA를 설치합니다.

## 로컬 테스트

Service Worker는 `file://`에서 정상 동작하지 않습니다. 아래처럼 로컬 서버에서 테스트하세요.

```bash
python -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속 후 F12 > Application > Service Workers 상태를 확인합니다.

## 주의

- 구형 `.xls` 직접 가져오기는 브라우저 오프라인 환경에서 안정성이 낮아 지원하지 않습니다.
- 기존 `.xls` 파일은 엑셀에서 `.xlsx` 또는 `.csv`로 다시 저장한 뒤 가져오는 방식이 안전합니다.
- V9.5 JSON 백업은 설정 화면의 `V9.5 JSON 마이그레이션`으로 변환할 수 있습니다.
