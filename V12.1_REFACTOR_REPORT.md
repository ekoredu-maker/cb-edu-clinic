# V12.1 Refactor Report

이번 단계는 V12의 단일 번들 위에 핵심 모듈을 추가하는 방식으로 진행했습니다.

## 반영 내용
- `assets/js/app.js` 유지: 기존 기능 호환층
- `assets/js/bootstrap.js` 추가: 모듈 부트스트랩
- `assets/js/core/store.js` 추가: 상태 조회/구독/패치 API
- `assets/js/data/persistence.js` 추가: 저장 레이어 래퍼
- `assets/js/domain/statistics.js` 추가: 통계 오버라이드 및 최신 인덱스 강제
- `assets/js/domain/verification.js` 추가: 검증 도메인 보강 및 최신 인덱스 강제
- `sw.js` 캐시명 `jc-edu-clinic-v12.1-cache`로 갱신

## 현재 성격
- 완전 재작성본이 아니라, 기존 번들을 호환층으로 두고 새 모듈을 얹은 2단 구조입니다.
- 가장 위험한 패턴인 후반 패치 체인을 점진적으로 대체하기 위한 중간 단계입니다.

## 다음 단계 제안
