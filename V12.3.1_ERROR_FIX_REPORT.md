# V12.3.1 오류 점검 보정 보고서

- 싱글 HTML: 하단 V12.3 패치 스크립트의 `const app` 중복 선언 제거
- 로그인: 기존 브라우저 저장소에 남은 비밀번호 해시로 인한 로그인 실패 대응용 `비밀번호 초기화(1111)` 버튼 추가
- 시간표 출력 UI: `var(--mute)` → `var(--muted)`, `btn-warn` → `btn-warning`, `btn-dark` → `btn-outline`
- GitHub/PWA 배포판: `manifest.webmanifest` V12.3.1 반영
- GitHub/PWA 배포판: `sw.js` 캐시 목록에 실제 런타임 JS 모듈 전체 반영
