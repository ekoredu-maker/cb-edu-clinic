# V11.1 패치 보고서

## 이번 수정
1. **실제 업로드 양식 기준 테스트 파일 재생성**
   - `학생_양식 (1).xlsx`
   - `지원단_양식.xlsx`
   의 헤더를 그대로 사용해 테스트 파일을 다시 생성했습니다.

2. **엑셀 업로드 영역 매핑 보강**
   - `지원영역(콤마)` 값 파싱 시 `label`, `altLabel`, `id`를 모두 허용하도록 수정했습니다.
   - 따라서 `한글미해득`처럼 양식 예시값으로 들어와도 정상 인식됩니다.

3. **실적검증 방식을 지원단 개인별 월단위 검증으로 변경**
   - 검증 화면이 날짜 나열형이 아니라 **지원단별 아코디언형 검증 UI**로 동작합니다.
   - 지원단별로 선택항목 일괄 승인 기능을 추가했습니다.

4. **중복 함수 혼란 최소화**
   - 기존 개인별 검증용 레거시 함수는 `_legacy...` 이름으로 바꾸고,
     최종 실행 함수는 `window.loadVerify`, `window.bulkVerifyByStf` 하나로 정리했습니다.
