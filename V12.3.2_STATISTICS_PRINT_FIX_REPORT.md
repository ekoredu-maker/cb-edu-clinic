# V12.3.2 Statistics & Print Fix Report

## Fixes

1. 통계 집계 기준 보정
   - 기존: `verified` / `paid` 상태만 통계 실적에 반영
   - 수정: `conducted` / `verified` / `paid` 상태를 통계 실적에 반영
   - 제외: `scheduled`, `canceled`, `rejected`
   - 월별 정산·집행액·지급명세서는 기존대로 승인(`verified`/`paid`) 기준 유지

2. 인쇄 출력 보정
   - 새 인쇄창에 원본 CSS를 복사할 때 기존 `@media print { body * { visibility:hidden } }` 규칙 때문에 출력물이 숨겨지던 문제 수정
   - 인쇄창 전용 CSS에서 모든 출력 대상의 `visibility`를 다시 보이도록 강제
   - 관리부, 시간표, 지급명세서, 집행내역서, 행정서식 인쇄 공통 안정화

3. 버전 표기
   - V12.3.2 Statistics & Print Fix Edition

## Notes

- 통계는 '입력된 실적' 기준입니다.
- 정산과 지급 서류는 행정 검증을 거친 승인 실적 기준입니다.
